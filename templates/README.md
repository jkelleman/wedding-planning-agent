## üîÑ How to Use Templates

### For Your Own Wedding:
1. **Copy the template** file you need
2. **Paste into your `/planning/` folder** (appropriate subfolder)
3. **Rename the file** (remove "_template" from filename)
4. **Customize with your details**
5. **Save and update as you plan**

### Example:
```
Copy: /templates/venue_and_catering/venue_cost_analysis_template.md
To: /planning/01_venue_and_catering/restaurant_name_cost_analysis.md
Then customize with restaurant's actual pricing
```

---

### To Share With Friends:
1. **Point them to this `/templates/` folder**
2. **Share the `REPOSITORY_STRUCTURE.md` file** so they understand the system
3. **They copy templates to their own workspace**
4. **They customize for their own wedding**

---

## üí° Tips for Using Templates

**Keep originals clean:**
- Never edit the template files directly
- Always copy first, then customize
- This way templates stay blank for reuse

**Customize freely:**
- Templates are starting points, not rules
- Add sections that matter to you
- Delete sections you don't need
- Make it work for your wedding style

**Create your own:**
- If you create a useful document during planning
- Save a blank "template" version
- Add it to the appropriate folder here
- Help future brides!

## üìã Template Checklist

When creating a new template, make sure it has:
- [ ] Clear descriptive filename ending in "_template"
- [ ] Instructions or comments explaining how to use it
- [ ] All personal details removed (replace with [Your Name], [Date], etc.)
- [ ] Section headers that are clear and self-explanatory
- [ ] Blank fields ready to fill in
- [ ] Examples or notes where helpful (in parentheses or comments)

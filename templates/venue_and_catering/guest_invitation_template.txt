## üí° Tips for Guest Management

### Before Sending Invitations
- [ ] Finalize complete guest list
- [ ] Get accurate addresses for all guests
- [ ] Decide on plus-one policy
- [ ] Create method for tracking RSVPs
- [ ] Set up wedding website if using one

### Invitation Best Practices
- [ ] Proofread everything multiple times
- [ ] Include RSVP deadline that's 2-3 weeks before event
- [ ] Make RSVP process easy (online, phone, or mail)
- [ ] Ask about dietary restrictions on RSVP card/form
- [ ] Include parking/transportation information
- [ ] Consider accommodation options for out-of-town guests

### RSVP Tracking
- [ ] Update spreadsheet/tracker as responses come in
- [ ] Note dietary restrictions immediately
- [ ] Follow up with non-responders 1 week after deadline
- [ ] Share final headcount with vendors by their deadline
- [ ] Create seating chart once you have final count

### Guest Experience
- [ ] Send pre-event reminder 1 week before
- [ ] Include detailed directions and parking info
- [ ] Designate point person for day-of questions
- [ ] Have emergency contact system for late/lost guests
- [ ] Consider welcome bags for out-of-town guests

### Post-Event
- [ ] Send thank you notes within 3 months
- [ ] Share photos with guests once received
- [ ] Return any borrowed items
- [ ] Pay outstanding vendor balances
